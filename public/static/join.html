Github

Der komplette Quellcode der Mod, sowie der Website ist OpenSource. Wir verwalten das Projekt über die Versionsverwaltungssoftware Git. Um dieses so einfach wie möglich zu gestalten Hosten wir unseren Quellcode über das Portal GitHub.

Programmieren

Jeder kann beim RewiMod helfen. Das einzige was du brauchst ist Anfängerwissen in Java. Hilfe beim Programmieren bekommst du im Forum oder auf unserem Discord Server. Für fragen über den Mod stehen dir dort profisionelle Entwickler zur verfügung.

Jeder kann helfen

Du kannst nicht programmieren, willst aber trotzdem helfen? Kein Problem! Hier ist genug Platz für alles rund um den RewiMod, egal ob du eine Fehler findest oder eine neue Idee hast. Villeicht motiviert dich das Projekt mit dem Programmieren anzufangen.
